## 📍 Introduction

markular is an open source **Mark**down editor for Ang**ular** >= 14.

## 💻 Installation

Install via npm:

```bash
npm i --save markular
```

## 🔨 Usage

Import the standalone component.

```ts
import { Markular } from 'markular';
```

and use it:

```html
<markular [(ngModel)]="markdown"></markular>
```

See the demo app for a more detailed example.
