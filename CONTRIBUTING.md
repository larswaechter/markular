# Contributing

Feel free to contribute to markular!

## Development setup

Build library:

```shell
npm run build
```

and run the demo app:

```shell
npm run serve
```

In order to run the tests enter:

```shell
npm test
```

## Contributing

1. Create a fork
2. Create a new branch from `release` (or `main` if not existing)
3. Make your changes
4. Add new tests if appropriated & run all tests
